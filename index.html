iConfRTC SDK for .NET


the iConfRTC SDK for .NET ( for windows forms and wpf ) is available for download from the Nuget gallery (x86 and x64 editions)

If you are not familiar with installing nuget packages from Visual Studio, here is a reference


You can find the iConfRTC SDK by searching for iconfrtc in the Nuget Package Manager or by using the Package Manager Console Install-Package command


32 bit versions


.NET Windows forms

https://www.nuget.org/packages/iConfRTC.Winform.X86/ 

                          PM> Install-Package iConfRTC.Winform.X86

.NET WPF

https://www.nuget.org/packages/iConfRTC.WPF.X86/ 

                          PM> Install-Package iConfRTC.WPF.X86

64 bit versions


.NET Windows forms

https://www.nuget.org/packages/iConfRTC.Winform.X64/ 

                          PM> Install-Package iConfRTC.Winform.X64

.NET WPF

https://www.nuget.org/packages/iConfRTC.WPF.X64/ 

                          PM> Install-Package iConfRTC.WPF.X64
